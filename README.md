# cooking
Приложение для поваров сетей ресторанов. Приложение позволяет авторезированным пользователям
создавать рабочие пространства, в которых они могут добавлять пользователей, создавать рецепты,
инвенторизировать ингридиенты а так же составлять задания для добавленных пользователей.
# Как запустить
откройте командную строку и клонируйте репозиторий
```commandline
git clone git@github.com:AlbertSabirzianov/cooking.git 
```
зайдите в каталог проекта
```commandline
cd cooking
```
создайте виртуальное окружение
```commandline
python -m venv venv
```
активируйте виртуальное окружение
```commandline
 source venv/Scripts/activate
```
загрузите необходимые библиотеки
```commandline
pip install -r src/requirements.txt 
```
откоректируйте секретный ключ токена в файле src/.env
```text
SECRET_KEY="your_secret_key"
```
запустите приложение
```commandline
uvicorn main:app 
```
документация будет доступна по адресу http://127.0.0.1:8000/docs# </br> <hr>

После запуска в директории src появится файл базы данных sqlite "cooking.db", в котором будут хранится все данные.
